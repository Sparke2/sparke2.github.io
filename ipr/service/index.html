СЕРВИС ДЛЯ ИНТЕГРАЦИИ ИНФОРМАЦИОННЫХ СИСТЕМ ОРГАНИЗАЦИЙ С IPR-SMART

С помощью разработанных функциональных возможностей вы можете настроить бесшовные переходы пользователей вашего портала в IPR SMART методом «клик по кнопке», а также «переход к изданию каталога IPR SMART».

Для подключения к сервису Вам необходимы 2 параметра:

  • специальный ключ
  • идентификатор портала, которые вы передадите в отдел разработки программного обеспечения вашей организации.

ВОЗМОЖНОСТИ ИНТЕГРАЦИИ

  • ДЛЯ ОСУЩЕСТВЛЕНИЯ ИНТЕГРАЦИИ С ПОРТАЛОМ НЕОБХОДИМО:
    1. Иметь право доступа — Администратор
    2. Сгенерировать ключ защиты и ID портала организации на странице «Модуль бесшовной авторизации».
  • ВАМ СТАНОВЯТСЯ
    ДОСТУПНЫ:
    1. Кнопка «Войти в IPR SMART», при клике на которую пользователь портала или АСУ перейдет в IPR SMART уже авторизованным
    2. Каталог изданий IPR SMART с возможностью перехода к изданиям в IPR SMART уже авторизованным
  • ПОЛУЧЕННЫЙ КОД УЧАСТВУЕТ В ФОРМИРОВАНИИ НОВОГО КЛЮЧА, ЗАВИСЯЩЕГО ОТ ДАТЫ ЕГО ГЕНЕРАЦИИ.

НА СТРАНИЦУ СЕРВИСА ПЕРЕДАЮТСЯ:

СИСТЕМНЫЕ ДАННЫЕ
  • ID портала из IPR SMART
  • ID пользователя портала
  • Дата в специальном формате YYYYMMDDHHMMSS
  • Уникальный каждый день ключ, сфомированный как MD5, $secret_key – код, полученный от нас.
ПОЛЬЗОВАТЕЛЬСКИЕ ДАННЫЕ
  • name
  • lastname
  • ut
  • email
  • bid – id издания в ЭБС, на который произойдет переадресация после авторизации

ПЕРЕХОДЫ АВТОРИЗОВАННЫХ ПОЛЬЗОВАТЕЛЕЙ

Все издания ЭБС имеют свой идентификатор, который может быть получен из адреса издания в IPR SMART.

Например, издание «Деловое общение» издательства «Дашков и К» имеет индекс в ЭБС 24780.

Таким образом, если в строке автоматической авторизации будет присутствовать &bid=24780, то после авторизации пользователь перейдет на это издание.

На портале возможно подготовить страницу, отвечающую за формирование адреса с учетом ID издания в ЭБС и адресующую пользователя после перехода по ссылке из каталога.
